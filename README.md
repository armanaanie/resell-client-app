# ♻️ ReSell Hub

A modern second-hand marketplace built with the MERN stack where users can buy, sell, and manage pre-owned products securely. ReSell Hub promotes sustainable shopping by giving products a second life while reducing waste.

---

## 🌐 Live URL

**Client:** https://resell-client-app.vercel.app/





---

## 🎯 Project Purpose

ReSell Hub is designed to provide a secure and user-friendly platform for buying and selling second-hand products. The platform supports multiple user roles, secure authentication, online payments, and comprehensive dashboards for buyers, sellers, and administrators.

By encouraging the reuse of products, ReSell Hub also helps reduce environmental waste and supports sustainable consumption.

---

## ✨ Key Features

- User authentication using Better Auth with JWT
- Role-based dashboards (Buyer, Seller, Admin)
- Add, edit, delete, and manage products
- Product search, filtering, sorting, and pagination
- Product wishlist functionality
- Secure online payment with Stripe
- Order management for buyers, sellers, and admins
- Product approval system for administrators
- Featured products section
- User profile management
- Sustainability Impact section promoting eco-friendly shopping
- Responsive design for desktop, tablet, and mobile devices
